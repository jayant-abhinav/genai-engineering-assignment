# Evaluation Report

## Overview

The RAG application was evaluated using a custom knowledge base consisting of eight technical documents covering Retrieval-Augmented Generation, embeddings, vector databases, ChromaDB, document chunking, prompt engineering, Large Language Models, and FastAPI.

A dataset of twenty evaluation questions was created to assess retrieval quality, answer generation, and overall response quality.

Answers generated by the RAG system were evaluated using the LLM Judge (Problem 2).
